HTML TAGS

Doctype defines the type of web page
head stores title and meta data of web page
title name of the webpage
body where all the information is displayed
bg color selects the color of a particular selected slot
h(1,2,3,4,5,6) change the size of the text
p tag write the text continously in a paragraph
table helps to create a table
iframes to add a popup of other website
img to add a image to your webpage
marquee to display a moving element(text)
